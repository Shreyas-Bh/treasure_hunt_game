🏝️ Treasure Hunt Game (Python)



A fun beginner-friendly text adventure game built in Python 🎮🐍



This project is a simple decision-based terminal game where the player goes on a mission to find hidden treasure on Treasure Island. The journey is full of choices, and each choice leads to a different outcome — victory, survival, or game over.



The game starts at a crossroads where the player must decide whether to go left or right. Making the correct choices through the island, shore, and mystery doors eventually leads to the treasure chest.



It’s a great beginner Python project that demonstrates:



✅ print() statements

✅ Multi-line ASCII art

✅ input() handling

✅ Nested if-elif-else conditions

✅ Decision-based game logic

✅ User interaction in terminal

✅ Basic storytelling with Python

🚀 How to Play

Run the Python file

Follow the on-screen story prompts

Enter your choices carefully:

left or right

wait or swim

red, blue, or yellow

Reach the correct path to win the treasure 🏆



💡 Learning Purpose



This project was created as part of Python practice to improve understanding of:

User input flow

Story-driven coding

Clean terminal output formatting



Perfect for beginners who are learning Python fundamentals through mini projects.
